# Claude Code Skills & Tools

A collection of custom skills, plugins, and agents for [Claude Code](https://claude.ai/claude-code).

## Skills

### Keyword Research
SEO keyword research skill — discover, analyse, and organise keywords for SEO campaigns. Includes competitor gap analysis, search intent classification, keyword clustering, and prioritisation using Ahrefs data.

### Schema Markup
Creates bespoke JSON-LD structured data and schema markup strategies for websites. Covers all page types (homepage, product, local business, article, etc.), AI visibility optimisation, and agentic commerce readiness.

## Installation

Copy any skill folder into your `.claude/skills/` directory:

```bash
cp -r skills/keyword-research ~/.claude/skills/
cp -r skills/schema-markup ~/.claude/skills/
```

Or for a project-specific installation:

```bash
cp -r skills/keyword-research your-project/.claude/skills/
cp -r skills/schema-markup your-project/.claude/skills/
```
