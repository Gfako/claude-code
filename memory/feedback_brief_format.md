Content briefs must follow the house style format — NO tables (except optionally in Recommendation). Use bullet lists with inline numbers like `- Keyword (Xk/month, not ranking)`. Strikethrough keywords to ignore. Keep analysis paragraphs short and opinionated. Write like a senior SEO strategist giving recommendations, not a tool dumping data.

**Why:** George shared a reference brief that uses this format. Tables make it harder to scan and feel too "report-like". The brief should be a working document for content creation, not a data dump.

**How to apply:** Every section of the content brief output. Especially Keywords (bullet list, not table), Off-page (one-liner per competitor), SERP analysis (numbered list), and Internal linking (simple slug list).
